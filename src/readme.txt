nombre del ejecutable:   S1.05.Java_Utils

1- navegar por directorios en la terminal con el comando cd seguido del directorio,
   en mi caso ha sido:

   > cd C:\Users\almam\IdeaProjects\S1.05.Java_Utils\src\main\java\Nivell2y3\Main

2- ejecutar dentro del anterior directorio el siguiente 
   comando para ejecutar el .java 

   > javac Main.java

3- volvemos hacia atrás en los directorios hasta llegar a la carpeta src con el comando 

   > cd ..

4- Ejecutar el archivo .class creado a partir del comando anterior
   con el siguiente comando:

   > java main.java.Nivell2y3.Main (poniendo esta vez puntos en lugar de \)

5- se habrá ejecutado el programa con todos los ejercicios juntos en uno
   se creara un .txt de todos los directorios, otro .txt de todos los archivos
   encpntrados de txt y el archivo .properties encriptado.
